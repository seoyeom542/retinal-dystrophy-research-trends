이 공간은 유전성 망막이영양증을 공부하면서 정리한 노트를 모아두는 **개인 연구
아카이브**입니다. 논문 요약, 개념 정리, 실험 아이디어 등 무엇이든 자유롭게 쌓아
갑니다.

## 노트를 추가하는 방법

새 노트는 마크다운 파일 하나만 만들면 됩니다.

1. `notes/` 폴더에 `YYYY-MM-DD-제목.md` 형식으로 파일을 만듭니다.
2. 파일 맨 위에 아래처럼 **머리말(frontmatter)** 을 적습니다.
3. `python src/build_notes.py` 를 실행하면 `docs/notes/` 에 HTML이 생성됩니다.
4. 커밋·push 하면 사이트에 올라갑니다.

```yaml
---
title: 노트 제목
date: 2026-06-10
tags: 태그1, 태그2
lang: ko
summary: 목록에 표시될 한 줄 요약.
---

여기부터 마크다운 본문을 씁니다.
```

## 마크다운으로 쓸 수 있는 것들

제목, 목록, **굵게**, *기울임*, `인라인 코드`, [링크](https://pubmed.ncbi.nlm.nih.gov/)는
물론, 표와 코드 블록, 인용도 됩니다.

| 유전자 | 연관 질환 | 비고 |
|--------|-----------|------|
| ABCA4 | 스타가르트병, CORD | 가장 흔한 IRD 원인 |
| RPE65 | LCA | 최초 FDA 승인 유전자치료 표적 |
| RPGR | X-연관 RP/CORD | ORF15 변이 호발 |

> 인용문도 이렇게 표시됩니다. 논문에서 인상 깊은 문장을 옮겨 적기 좋습니다.

```python
# 코드 블록도 문법 강조와 함께 렌더링됩니다.
genes = ["ABCA4", "RPE65", "RPGR"]
for g in genes:
    print(g)
```

## 이 아카이브를 쓰는 방식 (예시)

- **논문을 읽으면** → 핵심 요지·방법·결과를 노트로 요약
- **개념이 헷갈리면** → 내 언어로 다시 정리한 설명 노트
- **아이디어가 떠오르면** → 연구 가설·실험 설계 메모

노트가 쌓이면 [노트 목록](index.html)에서 최신순으로 모아 볼 수 있습니다. 본 프로젝트의
[대시보드](../index.html)·[논문 번역](../paper/index.html)과 함께, 공부의 흐름이
하나의 사이트에 기록됩니다.
